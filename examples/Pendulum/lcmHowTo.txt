
If you want to run the pendulum simulation demos over LCM, just launch 
three separate instances of matlab and run the following (one line in 
each instance):

>> runLCMDynamics(PendulumDynamics,PendulumLCMCoder)

>> runLCMVisualizer(PendulumVisualizer,PendulumLCMCoder)

>> runLCMControl(PendulumEnergyShaping(PendulumDynamics),PendulumLCMCoder)
